Built-in themes

PyKwiki features some built-in themes that can be installed with pykwiki theme install <theme_name>.

Default

The default PyKwiki theme.

Features

  • Valid HTML5
  • Mobile friendly
  • Uses Bootstrap 3
  • Supports searching
  • Includes respond.js for IE8 support
  • Includes mustache.js for javascript templates
  • Supports GitHub, Facebook, and Twitter: like, follow, watch.

Amelia

A colorful theme based on Amelia from Bootswatch.

Features

  • Valid HTML5
  • Mobile friendly
  • Uses Bootstrap 3
  • Supports searching
  • Includes respond.js for IE8 support
  • Built-in support for Github, Facebook, and Twitter social buttons

Barebones

A minimal stripped starter theme without CSS, designed to be used as a template for new themes.

Features

  • Infinitely nested menus
  • Valid HTML5
  • Ready for customization
  • No CSS
  • Supports searching
